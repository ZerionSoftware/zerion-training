What Do I Do?

Welcome to the final assignment of the Dataflow Automation Certification! In this assignment you will create a Dataflow to meet a variety of real-world business needs.

We will again be working with data submitted from the Residential Building Inspection form used in assignment 1.

Requirements

  • Add a property named 'timestamp', based on MODIFIED_DATE, in the format 2017-07-19 13:30:00
  • Add a property named 'date', based on the timestamp, in the format 07/19/2017
  • Add a property named 'time', based on the timestamp, in the format 1:45 PM
  • Add a property named 'full_state' converting abbreviation to state name
  • Flatten the record using flatteniFormObject()
  • Convert all keys using toKeysLowerCase()
  • Remove any properties that are null or an empty string
  • Add a branch for records created by the API
  • Add a branch for records missing a signature
