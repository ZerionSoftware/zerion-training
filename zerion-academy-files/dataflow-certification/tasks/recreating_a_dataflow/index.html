Task Instructions Dataflow Model Submit Task

What Do I Do?

In this certification task you will be required to recreate the Dataflow shown in the Dataflow Model tab. Begin by creating a new Dataflow and make sure your username is included in the name. Upon completion, export the Dataflow and in the Submit Task tab follow the instructions to upload your submission for review.

After submitting your Dataflow, check your email for the results. Any mistakes will be noted in the result email. When the task is successfully completed your certification progress will be updated.

The following requirements must be met:

  • Dataflow name includes your username
  • Same number of Dataflow Objects
  • Dataflow Object names must match exactly
  • Type of each Dataflow Object must be correct
  • RecordSets are linked correctly

List of Dataflow Object Names
  • wh_data_collector
  • full_records
  • parent_records
  • overall_count
  • subform_records
  • passing_scores
  • failing_scores
